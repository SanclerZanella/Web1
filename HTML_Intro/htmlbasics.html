What is HTML?

HTML stand for Hyper Text Markup Language. 'Hyper Text' means the document contains links to allow the user to navigate to other places in the document or to another document. 'Markup Language' is a means of providing in structions to a computer as to how it should process and present text, medias, etc. contained in the document. These instructions are provided using 'Tags' and 'Attributes'.

HTML Tags

Tags are used in HTML to 'mark-up' where an element begins and (where relevant) end. Tags are enclosed with '< >'. As you can see, the opening tag for this element looks like <p> and its closing tag looks </p>. Note the forward-slash in the closing tag.

HTML Attributes

We can use attributes when need to provide more information about an element. Attributes details including the attribute name and the associated information text, are specified inside an opening html tag. <tagname attributename=attributevalue>
